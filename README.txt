----------------------------------------------------------------Beat Pulse---------------------------------------------------
+ Giới thiệu
- Game được code bằng NLTT C# và sử dụng thư viện Unity


+ Mục đích 
- rèn luyện khả năng code
- giải trí, luyện phản xạ

Bắt đầu 

Nhấn nút Start game  ->  Bắt đầu trò chơi.  
Nhấn nút Spacebar để bắt đầu game 

*Cách chơi: 

Phím  Xanh lam  : Bấm nút D 
Phím  Đỏ: Bấm nút F 
Phím  Vàng: Bấm nút J 
Phím  Xanh lá: Bấm nút K     

  Bấm các Notes trên màn hình kịp lúc trước khi nó rơi qua các phím 

Trượt quá 10 Notes sẽ kết thúc game. 

Cách tính điểm: 1 Note (100 điểm) ; 1 Good Note ( 125 điểm ) ; 1 Perfect Note (150 điểm); 1 Missed Note ( 0 điểm) . 

*Kết thúc 

Hiện “ Result” 

Nhấn nút “Play Again” để chơi lại u.

+ Hướng dẫn cài đặt( lưu ý chỉ dành cho Window)
 B1 : tải thư mục này về  
 B2. Giải nén ra vào tìm file có tên Myproject.exe và Trải nghiệm thôi
